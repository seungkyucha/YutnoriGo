# 윳놀이GO — 게임 기획서 & 장르 분석

> 한국형 코인 루터(Coin Looter) — 모노폴리GO × 코인마스터 벤치마킹

---

## 1. 코인 루터(Coin Looter) 장르 분석

### 1.1 장르 정의
**코인 루터**는 "주사위/슬롯을 굴려 → 보드를 이동 → 재화를 획득 → 건설/수집으로 진척"하는 루프를 핵심으로 하는
캐주얼 수집형 장르다. 대표작은 **모노폴리GO**와 **코인마스터**이며, 두 게임 모두
"한 번의 액션(굴리기)"이 강렬한 시청각 보상으로 즉시 연결되는 **도파민 루프**에 모든 것을 건다.

### 1.2 모노폴리GO 핵심 포인트
| 요소 | 설명 | 윳놀이GO 적용 |
|------|------|---------------|
| **주사위 굴리기** | 제한된 주사위(roll) 자원, 시간/이벤트로 회복 | **윷 던지기**로 대체 (도·개·걸·윷·모·백도) |
| **보드 순회** | 정사각 보드를 돌며 칸마다 이벤트 | 모노폴리GO식 둘레형 보드(20칸) |
| **랜드마크 건설** | 보드를 클리어하며 도시(보드)를 완성 | 한국 도시 건설 (서울→부산→...) |
| **배수(Multiplier)** | 베팅 배수로 획득/소모를 증폭 → 고위험 고수익 | 배수 시스템 그대로 |
| **윷/모 보너스** | 더블 주사위 시 추가 굴리기 | 윷·모 시 **한 번 더** |
| **셔틀형 진척** | 보드 완성 → 다음 보드(도시) 잠금 해제 | 도시 완성 → 다음 도시 |

### 1.3 코인마스터 핵심 포인트
| 요소 | 설명 | 윳놀이GO 적용 |
|------|------|---------------|
| **슬롯 머신** | 굴리기 자체가 슬롯 → 공격/방어/약탈/코인 | 윷 결과 + **이벤트 칸** |
| **공격(Attack)** | 타 마을 시설 파괴로 코인 약탈 | **습격 칸** (AI 마을 약탈) |
| **방패(Shield)** | 공격 방어, 최대 3개 비축 | **방패 칸** |
| **약탈(Raid)** | 타 마을 땅을 파 코인 획득 | **대박/약탈 이벤트** |
| **빌리지 건설** | 코인으로 마을 시설 5개 건설 → 다음 마을 | 도시 랜드마크 건설 |

### 1.4 두 장르가 공유하는 "성공의 3원칙"
1. **즉각적·과장된 보상 연출** — 코인이 화면을 뒤덮고 카운터로 빨려 들어가는 연출이 핵심.
   → 윳놀이GO는 **윷이 공중에서 구르는 3D 물리 연출**과 **코인 폭발 → 상단 흡수 파티클**에 최대 역량 투입.
2. **짧고 반복 가능한 세션** — 굴리기 1회 = 3~8초, "한 번만 더"의 심리.
3. **수집 진척의 시각화** — 도시/랜드마크가 눈앞에서 완성되는 성취감.

## 2. 윳놀이GO 게임 디자인

### 2.1 한 줄 컨셉
> "윷을 던져 한국 방방곡곡을 건설하는 한국형 코인 루터"

### 2.2 코어 루프
```
윷 던지기(3D) → 결과(도/개/걸/윷/모/백도) → 말 이동
   → 도착 칸 이벤트 발동(코인/대박/습격/방패/세금/보너스)
   → 코인 폭발 연출 + 상단 카운터 흡수
   → 코인으로 랜드마크 건설 → 도시 완성 → 다음 도시
```

### 2.3 윷 결과 (전통 규칙)
| 결과 | 윷가락(평/배) | 이동 | 보너스 |
|------|--------------|------|--------|
| 백도 | 1개 평(뒷면 표시) | -1 | - |
| 도 | 1개 평 | +1 | - |
| 개 | 2개 평 | +2 | - |
| 걸 | 3개 평 | +3 | - |
| 윷 | 4개 평 | +4 | **한 번 더** |
| 모 | 0개 평(전부 배) | +5 | **한 번 더** |

### 2.4 보드 칸 구성 (20칸 둘레형)
- **출발(START)** — 통과/도착 시 코인 + 윷 보너스
- **코인 칸** (×2/×4/×6 배수) — 가장 흔함
- **랜드마크 건설 칸** — 현재 도시 랜드마크 건설 진행
- **대박(JACKPOT)** — 슬롯형 대량 코인
- **습격(ATTACK)** — AI 마을 약탈 미니 연출 + 코인 약탈
- **방패(SHIELD)** — 방패 +1 (최대 3)
- **보너스(BONUS)** — 윷 굴리기 +N
- **세금(TAX)** — 코인 소량 차감 (위기 연출)
- **이벤트(EVENT)** — 랜덤 (룰렛형)

### 2.5 도시 & 랜드마크 (건설 콘텐츠)
| 순서 | 도시 | 랜드마크 |
|------|------|----------|
| 1 | 서울 | 경복궁 · 광화문 · N서울타워 · 한강 · 동대문 |
| 2 | 부산 | 광안대교 · 해운대 · 감천문화마을 · 자갈치시장 |
| 3 | 경주 | 첨성대 · 불국사 · 석굴암 · 안압지 |
| 4 | 전주 | 한옥마을 · 전동성당 · 경기전 |
| 5 | 제주 | 성산일출봉 · 한라산 · 돌하르방 · 주상절리 |

각 랜드마크는 다단계 업그레이드(레벨업)되며, 모두 완성하면 도시 클리어 → 보상 + 다음 도시 잠금 해제.

### 2.6 재화 & 자원
- **코인(Coin)** — 메인 재화, 건설/배수에 사용
- **윷(Roll)** — 굴리기 자원, 시간 회복 + 보너스 칸
- **방패(Shield)** — 습격 방어
- **배수(Bet)** — 굴리기당 보상/비용 배수 (×1 ~ ×10)

## 3. 아트 & 연출 방향
- **스타일**: 스타일리쉬 3D 카툰 — 채도 높은 단청 컬러, 부드러운 그림자, 블룸(bloom).
- **최우선 연출 (Hero Moments)**:
  1. **윷 던지기** — 4개 윷가락이 공중에서 3D로 구르며 떨어지는 물리감 + 착지 임팩트.
  2. **코인 폭발 & 흡수** — 코인이 사방으로 터진 뒤 상단 카운터로 빨려 들어가는 3D 파티클.
- **한국 모티프**: 한옥 지붕, 청사초롱, 단청 문양, 태극 컬러, 전통 매듭.
- **2D UI**: SVG 기반 고해상도 스프라이트(버튼·아이콘·배너·패널).

## 4. 기술 스택
- **빌드**: Vite + TypeScript
- **3D**: Three.js + UnrealBloom 포스트프로세싱
- **UI**: 자체 제작 SVG 스프라이트 + CSS
- **저장**: LocalStorage
- **배포**: GitHub → Vercel (세로형 모바일 웹앱)
